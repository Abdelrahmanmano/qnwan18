Multi Deferred Accounts

Unlike standard Odoo, which uses a single global deferred account, this module enables assigning distinct deferred accounts to individual invoice or bill lines. Deferred revenue or expense accounts can also be configured per product, automatically applied upon selection. If none is specified, the default from Accounting settings applies.

Note: This app works only if your "Generate Expense/Revenue Entries" setting is set to "Bill/Invoice Validation".

Why Choose Our App?
